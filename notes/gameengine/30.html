1. Patching your version of PGE

  • you might want to install a patch to your version of PGE
    • this patch fixes a bug found when two circles collide at exactly the same time
  • to install this bug fix use the following commands

  • $ cd into your top level pge directory
    $     wget http://floppsie.comp.glam.ac.uk/download/pge/pge-feb28.diff
    $     wget http://floppsie.comp.glam.ac.uk/download/pge/pge-march01.diff
    $     wget http://floppsie.comp.glam.ac.uk/download/pge/localrun.sh.in
    $     patch -p1 < pge-feb28.diff
    $     patch -p1 < pge-march01.diff

  • notice the presence of localrun.sh this will appear in your build directory (after you have configured and compiled PGE)
  • localrun.sh can be used to run your Python code ( frozenbubble.py ) and will ensure that you pick up your local version of PGE rather than the system copy
  • to build PGE please examine previous notes
