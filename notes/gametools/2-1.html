20. Tricky example code 

  • #!/usr/bin/python3


for n in range(2, 10):
    print("n is", n)
    for x in range (2, n):
        print("x is", x)
        if n % x == 0:
            print(n, "equals", x, "*", n/x)
            break
    else:
        print(n, "is a prime number")

  • ./py6.py
2 is a prime number
3 is a prime number
4 equals 2 * 2
5 is a prime number
6 equals 2 * 3
7 is a prime number
8 equals 2 * 4
9 equals 3 * 3
