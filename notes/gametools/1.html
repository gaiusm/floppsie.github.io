14. Slicing

  • given a string, s= "hello world"
    • can obtain portion of string via: s[2:5]
    • yields: llo
  • first character has index 0
    • and also -11
    • last character index is 10 in this example
    • last character index is also -1
  • negative values start at right and move to the left
  • strings can be sliced using positive and negative values

21. Tricky example code 

  • #!/usr/bin/python3


for n in range(2, 10):
    print("n is", n)
    for x in range (2, n):
        print("x is", x)
        if n % x == 0:
            print(n, "equals", x, "*", n/x)
            break
    else:
        print(n, "is a prime number")

  • ./py6.py
2 is a prime number
3 is a prime number
4 equals 2 * 2
5 is a prime number
6 equals 2 * 3
7 is a prime number
8 equals 2 * 4
9 equals 3 * 3
